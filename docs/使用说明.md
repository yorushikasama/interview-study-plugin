# Interview Study 使用说明

本文档适用于 Interview Study `0.3.3`。插件已在 IntelliJ IDEA `2025.1.2` 上开发和验证。

## 1. 插件功能

Interview Study 在 IDEA 右侧提供一个面试学习工具窗口，包含三个页签：

- **生成题目**：根据目标岗位或本地资料调用 AI 自动混合生成问答题和单选题。
- **题库练习**：先自行回答，再查看参考答案、AI 批改、追问，并安排后续复习。
- **设置**：填写 OpenAI 兼容接口的 Base URL、API Key 和模型名。

插件不需要账号和独立服务器。题库与复习状态保存在本机 IDEA 配置中，API Key 保存到 JetBrains Password Safe。

## 2. 安装插件

### 2.1 找到安装包

当前安装包位于：

```text
dist\interview-study-plugin-0.3.3.zip
```

安装时直接选择 ZIP 文件，不要解压。

### 2.2 从 IDEA 安装

1. 打开 IntelliJ IDEA。
2. 进入 `File | Settings`。
3. 选择左侧的 `Plugins`。
4. 点击插件页面右上角的齿轮按钮。
5. 选择 `Install Plugin from Disk...`。
6. 选择 `interview-study-plugin-0.3.3.zip`。
7. 点击确认并按提示重启 IDEA。

### 2.3 打开插件

重启后，IDEA 右侧工具栏会显示 `Interview Study`。

如果右侧没有显示：

1. 打开 `View | Tool Windows`。
2. 选择 `Interview Study`。

## 3. 配置 AI 接口

打开 `Interview Study | 设置`，填写以下内容：

| 配置项 | 说明 |
| --- | --- |
| `Base URL` | OpenAI 兼容接口地址，例如服务商提供的 API 根地址。可以填写带 `/v1` 或不带 `/v1` 的地址。 |
| `API Key` | 服务商创建的 API Key。插件使用 JetBrains Password Safe 保存，不写入项目文件。 |
| `模型名` | 服务商支持的模型标识，必须与接口返回或服务商文档中的名称完全一致。 |

配置完成后：

1. 点击 `保存`。
2. 点击 `测试连接`。
3. 出现“连接成功”后即可生成题目。

注意：测试连接会实际调用一次 AI 接口，可能产生少量费用。

## 4. 根据岗位生成题目

1. 打开 `生成题目 | 按岗位生成`。
2. 填写目标岗位，例如 `Java 后端`。
3. 填写级别，例如 `中级`、`高级`。
4. 填写技术栈，例如 `Java、Spring Boot、MySQL、Redis`。
5. 填写出题方向，例如 `并发、项目追问、故障排查`。
6. 设置题目数量，范围为 `1–20`。
7. 点击 `AI 生成题目`。

生成成功后，题目会自动保存到本地题库，并出现在 `题库练习` 页签中。题型由 AI 根据内容自动决定，当前只保留问答题和单选题；不需要手动选择题型。

## 5. 根据资料生成题目

支持以下文件格式：

- PDF：`.pdf`
- Word：`.docx`
- Markdown：`.md`、`.markdown`
- 纯文本：`.txt`

使用步骤：

1. 打开 `生成题目 | 按资料生成`。
2. 点击 `选择资料`。
3. 等待插件提取文本，可点击 `预览文本` 检查解析结果。
4. 状态栏会显示资料字数、处理模式和预计 AI 调用次数。
5. 填写出题方向。
6. 设置题目数量。
7. 点击 `根据资料生成`。
8. 大资料会弹出费用确认提示，确认后才开始调用 AI。

### 5.1 小资料

不超过 `18,000` 字时，插件直接使用一次 AI 调用生成题目。

### 5.2 大资料

超过 `18,000` 字时，插件自动执行全文处理：

1. 按段落切分资料，每块最多约 `12,000` 字。
2. 逐块调用 AI 提炼知识卡。
3. 合并知识卡。
4. 再调用一次 AI 生成最终题目。

预计调用次数为：

```text
分块数量 + 1 次最终出题
```

插件最多处理 `20` 个分块，约 `24 万字`。超过限制会停止处理并显示错误，避免产生无上限费用。

### 5.3 资料解析限制

- 扫描版或图片型 PDF 没有可复制文本时无法解析；当前版本不包含 OCR。
- DOCX 当前主要读取普通段落，复杂表格、文本框、图片内容可能无法提取。
- Markdown 和 TXT 按 UTF-8 文本读取。
- 大资料开始调用后，当前版本不支持中途取消或断点续传。
- 任一分块处理失败时不会保存半成品题目，可以修正接口问题后重新生成。

## 6. 练习和复习

1. 打开 `题库练习`。
2. 可按关键词、题型、标签、来源筛选，也可只看 `今日复习`、`收藏`、`错题`。
3. 从左侧选择一道题，在作答区域先写自己的回答。
4. 点击 `显示参考答案` 进行对照，或点击 `AI 批改` 获取评分、缺口和改进答案。
5. 点击 `AI 追问` 可基于当前题生成更深入的追问。
6. 根据掌握情况选择：
   - `不会`：标记错题，复习间隔回退。
   - `会了`：逐步延长复习间隔，依次约为 2、4、7、14、30 天。
   - `已掌握`：直接进入掌握状态。
7. 可用 `收藏/取消`、`错题/取消` 管理重点题；可用 `导出`、`导入` 备份或迁移 JSON 题库。

题目列表会显示复习状态，下方统计会显示总数、今日待复习、今日已练和掌握率。

## 7. 数据和隐私

- API 请求会把岗位描述、技术栈或资料文本发送给用户配置的 AI 服务商。
- 不要上传包含密码、密钥、客户隐私或其他敏感信息的资料。
- API Key 存储在 JetBrains Password Safe。
- Base URL、模型名、题库和复习状态保存在 IDEA 本机配置中。
- 当前题库属于 IDEA 应用级数据，在不同项目窗口之间共享。

## 8. 更新插件

安装新版本时：

1. 下载或构建新的插件 ZIP。
2. 再次进入 `Settings | Plugins`。
3. 选择 `Install Plugin from Disk...`。
4. 选择新版本 ZIP。
5. 重启 IDEA。

只要插件 ID 不变，本地题库和设置会继续沿用。更新前仍建议备份重要题目。

## 9. 卸载插件

1. 进入 `Settings | Plugins | Installed`。
2. 搜索 `Interview Study`。
3. 点击 `Uninstall`。
4. 重启 IDEA。

卸载插件不保证自动删除 IDEA 配置目录中已经保存的数据。

## 10. 常见问题

### 10.1 测试连接返回 401 或 403

通常表示 API Key 无效、没有权限、额度不足，或服务商要求额外授权。请重新检查 Key 和服务商账户状态。

### 10.2 返回 404

检查 Base URL 是否正确。插件会自动在未带版本号的地址后补充 `/v1/chat/completions`，因此不要填写完整的 `/chat/completions` 地址。

### 10.3 提示模型不存在

模型名必须与服务商支持的模型标识完全一致。不要填写模型的展示名称或中文名称。

### 10.4 AI 返回格式不正确

插件要求模型返回 JSON。部分能力较弱或不兼容的模型可能不遵守格式，可以重试或更换指令遵循能力更好的模型。

### 10.5 选择 PDF 后提示没有文本

文件可能是扫描版 PDF。请先使用 OCR 软件将其转换成可搜索 PDF，或导出为 DOCX/TXT 后再使用。

### 10.6 大资料调用次数过多

减少资料范围，删除无关章节，或将大文件拆成多个主题文件分别生成。当前版本优先保证全文覆盖，不做向量检索和相关段落筛选。

### 10.7 生成过程中关闭 IDEA

尚未完成的题目不会保存。重新打开 IDEA 后需要再次生成。

## 11. 从源码构建

本机需要使用 IDEA 2025.1.2 自带的 JBR 21：

```powershell
cd D:\interview\interview-study-plugin
$env:JAVA_HOME='D:\IntelliJ IDEA 2025.1.2\jbr'
.\gradlew.bat buildPlugin --offline
```

构建产物位于：

```text
build\distributions\
```

首次构建如果本地没有 Gradle 和依赖缓存，需要联网下载构建依赖。
